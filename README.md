# 🏭 OptiProd AI – Predictive Manufacturing & Downtime Optimization Portal

OptiProd AI is an entry-level, industry-aligned **Predictive Maintenance & Production Optimization Portal** designed for smart factory ecosystems. By leveraging machine learning and interactive telemetry simulations, it empowers production managers to monitor IoT sensor feeds, predict equipment failures before they happen, and drastically reduce costly, unplanned factory downtime.

---

## 🚀 Key Features

*   **AI Breakdown Predictor:** Uses an trained Machine Learning model to evaluate multi-variable sensor arrays and return exact real-time failure probabilities.
*   **Live Telemetry Simulator:** Interactive control panel allowing users to simulate fluctuating operational environments (Temperature, Vibration, and Tool Wear).
*   **Dynamic Operations Dashboard:** Sleek, high-fidelity UI tracking factory health metrics and dynamic warning levels (Optimal, Caution, Critical).
*   **Automated Action Protocol:** Triggers automated industrial recommendations instantly when risk probabilities exceed critical thresholds.

---

## 🛠️ Tech Stack

*   **Language:** Python
*   **Machine Learning:** Scikit-Learn (Random Forest Classification), Joblib
*   **Core Data Analytics:** Pandas, NumPy
*   **Frontend Web Framework:** Streamlit
*   **Data Visualization:** Plotly (Interactive Gauges & Telemetry Charts)
